# addTag

<Version version="8.37.0" />

## Overview {#overview}

Use the `addTag` command to add a tag (or multiple tags) to a test during execution.

## Usage {#usage}

```typescript
await browser.addTag("one");
await browser.addTag(["first", "second"]);
```
