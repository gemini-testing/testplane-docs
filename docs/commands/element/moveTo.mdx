# moveTo

## Overview {#overview}

Use the `moveTo` command to move the mouse by specified offsets along the X and Y axes relative to the specified element.

If no element is specified, the movement will be executed relative to the current mouse cursor position.
If an element is specified and no offsets are provided, the mouse will move to the center of the element.
If the element is not visible, it will scroll into view.

## Usage {#usage}

```javascript
await browser.$(selector).moveTo({ xOffset, yOffset });
```
