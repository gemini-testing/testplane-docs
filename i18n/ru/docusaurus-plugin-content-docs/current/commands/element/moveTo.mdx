# moveTo

## Обзор {#overview}

Используйте команду `moveTo`, чтобы сместить мышь на заданные расстояния по осям X и Y относительно указанного элемента.

Если элемент не указан, то перемещение будет выполнено относительно текущего курсора мыши.
Если элемент указан, а смещения по осям не указаны, то мышь будет перемещена в центр элемента.
Если элемент не виден, то будет выполнен подскролл так, чтобы элемент стал виден.

## Использование {#usage}

```javascript
await browser.$(selector).moveTo({ xOffset, yOffset });
```
