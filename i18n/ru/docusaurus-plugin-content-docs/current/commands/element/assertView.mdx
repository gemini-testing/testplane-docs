# assertView

## Обзор {/* #overview */}

Используйте команду `assertView`, чтобы снять скриншот элемента для определенного состояния теста и сравнить его с эталонным.

## Использование {/* #usage */}

```javascript
await browser.$(selector).assertView(state, options);
```

<Admonition type="info">
    Команда `assertView` использует [waitForExist][wait-for-exist] (с
    [waitTimeout][wait-timeout] и [waitInterval][wait-interval]), чтобы дождаться появления элемента перед снятием скриншота.

    `assertView` автоматически ждет загрузки статики (шрифтов, изображений и т.д.) с помощью команды [waitForStaticToLoad][wait-for-static-to-load].

</Admonition>

## Примеры использования {/* #examples */}

**example.js**

```javascript
it("should assert view", async ({ browser }) => {
    await browser.url("some/url");

    const elem = await browser.$(".button");

    await elem.assertView("plain");
    await elem.click();
    await elem.assertView("clicked");
});
```
