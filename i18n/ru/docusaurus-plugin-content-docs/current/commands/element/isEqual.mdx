# isEqual

## Обзор {#overview}

Используйте команду `isEqual`, чтобы сравнить заданный DOM-элемент с другим элементом.

Команда возвращает `true`, если выбранный элемент совпадает с предоставленным, иначе возвращает `false`.

## Использование {#usage}

```javascript
await browser.$(selector).isEqual(el);
```

## Примеры использования {#examples}

```javascript
it("should detect if an element is clickable", async ({ browser }) => {
    const el = await browser.$("#el");

    const sameEl = await browser.$("#el");
    const anotherEl = await browser.$("#anotherEl");

    el.isEqual(sameEl); // выведет: true
    el.isEqual(anotherEl); // выведет: false
});
```
