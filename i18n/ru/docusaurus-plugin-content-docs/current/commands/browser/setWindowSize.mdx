# setWindowSize

## Обзор {#overview}

Используйте команду `setWindowSize`, чтобы изменить внешний размер окна браузера в соответствии с заданными шириной и высотой.

## Использование {#usage}

```javascript
await browser.setWindowSize(width, height);
```

<tr><td>width</td><td>Number</td><td>Ширина окна браузера, которую нужно установить.</td></tr>
<tr><td>height</td><td>Number</td><td>Высота окна браузера, которую нужно установить.</td></tr>
