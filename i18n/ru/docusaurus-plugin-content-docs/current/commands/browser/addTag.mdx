# addTag

<Version version="8.37.0" />

## Обзор {#overview}

Используйте команду `addTag`, чтобы добавить один или несколько тегов к тесту во время его выполнения.

## Использование {#usage}

```typescript
await browser.addTag("one");
await browser.addTag(["first", "second"]);
```
