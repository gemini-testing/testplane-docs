# savePDF

## Overview {#overview}

Use the `savePDF` command to save the current browser context to a PDF file.

## Usage {#usage}

```javascript
await browser.savePDF(filepath, {
    orientation,
    scale,
    background,
    width,
    height,
    top,
    bottom,
    left,
    right,
    shrinkToFit,
    pageRanges,
});
```

## Command Parameters {#parameters}

<table>
<thead>
<tr><td>**Name**</td><td>**Type**</td><td>**Description**</td></tr>
</thead>
<tbody>
<tr><td>filepath</td><td>String</td><td>Path to the PDF file relative to the execution directory (the _.pdf_ extension is mandatory).</td></tr>
<tr><td>orientation</td><td>String</td><td>Orientation of the PDF page.</td></tr>
<tr><td>scale</td><td>Number</td><td>Scale of the PDF page.</td></tr>
<tr><td>background</td><td>Boolean</td><td>Include the background in the PDF.</td></tr>
<tr><td>width</td><td>Number</td><td>Width of the PDF page.</td></tr>
<tr><td>height</td><td>Number</td><td>Height of the PDF page.</td></tr>
<tr><td>top</td><td>Number</td><td>Top margin of the PDF page.</td></tr>
<tr><td>bottom</td><td>Number</td><td>Bottom margin of the PDF page.</td></tr>
<tr><td>left</td><td>Number</td><td>Left margin of the PDF page.</td></tr>
<tr><td>right</td><td>Number</td><td>Right margin of the PDF page.</td></tr>
<tr><td>shrinkToFit</td><td>Number</td><td>Shrink the page to fit the PDF page size.</td></tr>
<tr><td>pageRanges</td><td>Object[]</td><td>Range of pages to include in the PDF.</td></tr>

</tbody>
</table>

## Usage Examples {#examples}

```javascript
it("should save a PDF screenshot of the browser view", async ({ browser }) => {
    await browser.savePDF("./some/path/screenshot.pdf");
});
```
