# scroll

## Overview {#overview}

Use the `scroll` command to scroll the browser viewport. Note that the `x` and `y` coordinates refer to the current scrolling position of the viewport.

## Usage {#usage}

```typescript
await browser.scroll(x, y);
```

## Command Parameters {#parameters}

<table>
<thead>
<tr><td>**Name**</td><td>**Type**</td><td>**Description**</td></tr>
</thead>
<tbody>
<tr><td>x</td><td>Number</td><td>Optional parameter. The x-coordinate to scroll to. Default: _0_.</td></tr>
<tr><td>y</td><td>Number</td><td>Optional parameter. The y-coordinate to scroll to. Default: _0_.</td></tr>

</tbody>
</table>

## Usage Examples {#examples}

```typescript
it("should demonstrate the scroll command", async ({ browser }) => {
    await browser.url("https://webdriver.io");

    console.log(await browser.execute(() => window.scrollY)); // returns 0
    await browser.scroll(0, 200);
    console.log(await browser.execute(() => window.scrollY)); // returns 200
});
```
