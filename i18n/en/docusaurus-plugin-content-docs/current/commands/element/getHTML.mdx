# getHTML

## Overview {#overview}

Use the `getHTML` command to retrieve the HTML source code of a specified DOM element.

## Usage {#usage}

```javascript
await browser.$(selector).getHTML(includeSelectorTag);
```

## Command Parameters {#parameters}

<table>
<thead>
<tr><td>**Name**</td><td>**Type**</td><td>**Description**</td></tr>
</thead>
<tbody>
<tr><td>includeSelectorTag</td><td>Boolean</td><td>Optional parameter. Include the selector element's tag in the source code. Default: _true_.</td></tr>

</tbody>
</table>

## Usage Examples {#examples}

**index.html**

```html
<div id="test">
    <span>Lorem ipsum dolor amet</span>
</div>
```

**getHTML.js**

```javascript
it("should get html for certain elements", async ({ browser }) => {
    var outerHTML = await browser.$("#test").getHTML();
    console.log(outerHTML);
    // outputs:
    // "<div id="test"><span>Lorem ipsum dolor amet</span></div>"

    var innerHTML = await browser.$("#test").getHTML(false);
    console.log(innerHTML);
    // outputs:
    // "<span>Lorem ipsum dolor amet</span>"
});
```
